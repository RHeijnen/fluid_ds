Dark mode is a scheme switch, not a separate theme. `@fluid-ds/tokens` ships a
`light.css` and a `dark.css`; the dark sheet re-declares the surface, text, and
border semantics under `[data-fluid-theme="dark"]`. Set that attribute and
every component re-resolves, no dark variants, no duplicate styles.

## Setup

Load both scheme sheets after `base.css`, then set the attribute on `<html>`:

```html
<link rel="stylesheet" href="@fluid-ds/tokens/base.css" />
<link rel="stylesheet" href="@fluid-ds/tokens/light.css" />
<link rel="stylesheet" href="@fluid-ds/tokens/dark.css" />
```

```ts title="theme-toggle.ts"
const root = document.documentElement;

function setScheme(scheme /* "light" | "dark" */) {
  root.dataset.fluidTheme = scheme;
  localStorage.setItem("theme", scheme);
}

// Restore the saved choice, falling back to the OS preference.
const saved = localStorage.getItem("theme");
const prefersDark = matchMedia("(prefers-color-scheme: dark)").matches;
setScheme(saved ?? (prefersDark ? "dark" : "light"));
```

<Aside type="tip" title="Avoid the flash">
  Apply the attribute in a tiny inline `<script>` in `<head>`, before your CSS
  paints, so a returning dark-mode visitor never sees a white flash. (This docs
  site does exactly that.)
</Aside>

## It composes with everything

Because dark mode only moves the semantic surface/text tokens, your
[brand](/theming/brand/), [component](/theming/tokens/), and
[per-element](/theming/per-element/) overrides keep applying on top. The same
panel, light and dark:

The attribute works on **any** ancestor, not just `<html>`, which is exactly
how the demo above shows a dark panel on a light page. Scope it to a section if
you only want part of the UI dark.

## Scheme vs brand

Two independent axes you can combine freely:

- `data-fluid-theme` = **light / dark** (the surface + text scheme).
- `data-fluid-brand` = **which accent palette** (Default / Midnight / Corporate / your own).

So "Midnight brand in dark mode" is just both attributes set. They never fight,
one picks the canvas, the other picks the accent.
