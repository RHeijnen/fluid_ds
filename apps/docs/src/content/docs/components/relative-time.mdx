`<fluid-relative-time>` formats a date as a relative phrase ("5 minutes ago",
"in 2 days", "yesterday") using `Intl.RelativeTimeFormat`. It picks the largest
sensible unit automatically and re-renders once a minute so labels stay current.
For absolute dates, reach for [`<fluid-format-date>`](/components/format-date/).

<Demo>
  <fluid-relative-time date="2026-05-29T13:00:00Z"></fluid-relative-time>
</Demo>

### Numeric mode

`auto` (default) lets the formatter use phrases like "yesterday" and
"tomorrow". `always` forces the number form ("1 day ago").

```html
<fluid-relative-time date="…" numeric="always"></fluid-relative-time>
```

### Disable auto-refresh

```html
<fluid-relative-time date="…" no-sync></fluid-relative-time>
```

Set this for pages with hundreds of timestamps where you'd rather batch
refresh manually.

## When to use

- Activity feeds, message timestamps, anywhere "when did this happen?"
  is more useful as a relative phrase than an exact date.
- For an exact date you want to display alongside or instead, use
  [`<fluid-format-date>`](/components/format-date/).

## Accessibility

- The text is plain: screen readers announce it like any other word.
- For audit log / activity-feed use, pair the relative phrase with an
  exact timestamp (in a `<time datetime="…">` or as a tooltip) so users
  can disambiguate when needed.
