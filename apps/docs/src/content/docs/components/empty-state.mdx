`<fluid-empty-state>` is the friendly placeholder for "nothing here yet": an
optional icon or illustration, a heading, supporting text, and call-to-action
buttons, centered in a column.

<Demo>
  <fluid-empty-state heading="No projects yet">
    <fluid-icon slot="media" name="folder"></fluid-icon>
    Create your first project to get started.
    <fluid-button slot="actions" variant="primary">New project</fluid-button>
  </fluid-empty-state>
</Demo>

## Slots

- **`media`** an icon or illustration above the heading.
- **default** the supporting description text.
- **`actions`** call-to-action buttons.

```html
<fluid-empty-state heading="No results found">
  <fluid-icon slot="media" name="search"></fluid-icon>
  Try a different search term or clear your filters.
  <fluid-button slot="actions" variant="secondary">Clear filters</fluid-button>
</fluid-empty-state>
```
