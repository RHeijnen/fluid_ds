`<fluid-stat>` shows a single key metric: a label, a prominent value, and an
optional change with a trend direction. The change text always carries the
delta (e.g. `+12%`), so direction is never signalled by color alone.

<Demo>

    <fluid-stat label="Churn" value="2.1%" change="-0.4pt" trend="down"></fluid-stat>
  </div>
</Demo>

## Examples

```html
<fluid-stat label="Active users" value="9,310" change="+4%" trend="up"></fluid-stat>
```

`trend` is one of `up`, `down`, or `neutral`; it sets the arrow (↑ ↓ →) and the
color. Override `--fluid-stat-up-fg` / `--fluid-stat-down-fg` per brand.

## Accessibility

Each tile is a labelled `role="group"` summarising label, value, and change, so
screen readers announce the metric as a unit. Trend is conveyed by the arrow and
the signed change text, not color alone (SC 1.4.1).
