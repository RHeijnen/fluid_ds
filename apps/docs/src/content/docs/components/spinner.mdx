`<fluid-spinner>` is the indeterminate loader, a spinning arc you reach for when
you don't know how long an operation will take. It sizes from `font-size` (so it
scales inline with text) and respects `prefers-reduced-motion` by slowing the
rotation rather than removing it.

<Demo>
  <fluid-spinner style="font-size: 2rem;"></fluid-spinner>
</Demo>

### Sizing

The spinner inherits `font-size`. Set it on the host or a wrapper.

<Demo>
  <fluid-spinner style="font-size: 1rem;"></fluid-spinner>
  <fluid-spinner style="font-size: 1.5rem;"></fluid-spinner>
  <fluid-spinner style="font-size: 2.5rem;"></fluid-spinner>
</Demo>

### Coloring

`--fluid-spinner-color` controls the foreground arc; the background
track has its own token.

<Demo>
  <fluid-spinner style="font-size: 2rem; --fluid-spinner-color: #e11d48;"></fluid-spinner>
  <fluid-spinner style="font-size: 2rem; --fluid-spinner-color: #16a34a;"></fluid-spinner>
</Demo>

### Slower rotation

```html
<fluid-spinner style="--fluid-spinner-speed: 1.6s;"></fluid-spinner>
```

## When to use

- Operations of unknown length: saving, fetching, submitting.
- For known progress, use
  [`<fluid-progress-bar>`](/components/progress-bar/) or
  [`<fluid-progress-ring>`](/components/progress-ring/).
- For "loading the page structure", use
  [`<fluid-skeleton>`](/components/skeleton/) so the surrounding
  layout doesn't shift when content arrives.

## Theming

Every styled property reads a **component-scoped `--fluid-spinner-*` token that
falls back to a main semantic var** (the override ladder). Size inherits from
`font-size`. The **complete list** is in the [API table below](#api).

```css title="brand.css"
[data-fluid-brand="custom"] {
  --fluid-spinner-color: var(--fluid-accent-base);
  --fluid-spinner-track-color: var(--fluid-color-neutral-200);
  --fluid-spinner-speed: 1s;
}
```

### Beyond tokens: `::part()`

The SVG is [`part="base"`](#api). Reach it with
[`::part()`](https://developer.mozilla.org/en-US/docs/Web/CSS/::part) for CSS
not covered by a token.

## Accessibility

The cross-component contract is in the
[Accessibility guide](/guides/accessibility/); the notes below are
spinner-specific.

### Names & roles

- The host is `role="progressbar"` with `aria-label="Loading"` by default:
  override with a more specific label ("Saving…", "Loading results")
  ([SC 4.1.2](https://www.w3.org/WAI/WCAG22/Understanding/name-role-value)).
- For a known duration, prefer [`<fluid-progress-bar>`](/components/progress-bar/)
  or [`<fluid-progress-ring>`](/components/progress-ring/) so the value is
  conveyed, not just "busy".

### Motion

Honors `prefers-reduced-motion`, the rotation slows to ~6s so the spinner still
signals activity without rapid motion.
